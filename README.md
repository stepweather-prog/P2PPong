# P2PPong

Транспортный арбитр для RobinHood P2P. Бета-версия.

P2PPong обеспечивает установку защищённых WebRTC-соединений между пользователями RobinHood P2P через обмен SDP-пакетами на Render-сервере. Не хранит сообщения, не участвует в переписке. Только создаёт и верифицирует каналы связи.

## Условия использования

P2PPong является служебным компонентом RobinHood P2P. На него распространяются все условия использования RobinHood P2P. Используя P2PPong, вы соглашаетесь с условиями RobinHood P2P.

Автор не несёт ответственности за использование P2PPong третьими лицами в любых целях.
